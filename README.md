# Daily Atlas

A local-first everyday tracker for tasks, schedule, sleep, meals, workouts, study, work, metrics, rituals, and flexible activity logging.

Open `index.html` in a browser. Data is stored in the browser with `localStorage`.

To install it like an app on desktop or phone, host the folder on an HTTPS site such as GitHub Pages or Netlify, then use the browser's Install/Add to Home Screen option.
